How to build build a simple image recognition app ?

Let's start coding !

First, we will use the terminal to create a React.js application called image-recognition using the create-react-app package:

Now, inside out App.js file, we will remove all the unnecessary code to only render a title called Image classification with ML5.js.

Second Step - Choose the picture

The image that we are going to classify is this amazing picture of a tiger.
Download the image at https://unsplash.com/photos/5RBXc7R-YWs , rename if tiger.jpg and put it in your src folder.

Once you’re done, we will display the image inside our application just below our h1 tag:

Third Step - Getting started with ml5.js

When it’s done, we will start to implement ml5.js. This library will enable our app to recognize what’s on the image we’ve just downloaded.
First, install the ml5.js library using the terminal:

Once installed, we will import everything from the ml5.js library. After that, we will create a function called classifyImg inside the App component that will:

  1. Initialize the Image Classifier method with MobileNet.
  2. Load the model.
  3. Put the image to classify inside a variable called image.
  4. Get the 5 model’s predictions and output them in the console.

After creating the classifyImg function, we are going to call it inside the ComponentDidMount lifecycle method:

You should see in your console an array of 5 predictions objects sorted by probability. The first predictions should be “tiger, Panthera tigris” which is a pretty accurate guess.
Once we have the 5 predictions, we are going to save them in the state of the App component. To do this, we need to:

  1. Initialize the App component state with a predictions key with an empty array as value.
  2. Create a function called setPredictions that take the model predictions array as argument and updated the predictions state (previously an empty array) with those 5 predictions.
  3. Instead of printing the result in the console, we are going to return the predictions and use Promises chaining to call the newly created setPredictions function with the model predictions as parameters.

Final Step - Showing the result in the browser!

Now that we have the predictions inside our App component state, the last thing we have to do is to output the predictions to the user.
To do so, inside the render method, we are going to create a predictions variable that will store a default value while loading (empty div for now).
Once the model have made his predictions and that they’re stored in the state, we are going to map over the predictions array to display (1) the model’s prediction and (2) the probability for each of the 5 predictions.
Finally, we are going to display the predictions inside our app, just below our tiger image:
